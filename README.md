### 一、相关技术
- 前端：`Vue`、`微信小程序`、`HTML` / `CSS` / `Javascript` 。
- 后端：`Springboot`、`Java`、`JavaWeb` 。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`微信小程序开发工具`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：学生、教师、管理员。

学生功能：1. 登录、2. 注册、3. 首页、4. 课程、5. 论坛、6. 个人中心

教师功能：1. 登录、2. 首页、3. 个人中心、4. 用户管理、5. 课程管理、6. 学习足迹管理、7. 论坛管理、8. 公告信息管理

管理员功能：1. 登录、2. 首页、3. 个人中心、4. 管理员管理、5. 用户管理、6. 教师管理、7. 课程管理、8. 学习足迹管理、9. 基础数据管理、10. 论坛管理、11. 公告信息管理、12. 轮播图信息
